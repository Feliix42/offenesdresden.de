Code Week EU 2015in Dresden

Call For Participation - Was kannst Du zur Code Week tun?

Idee

2015 findet die Code Week in der Woche vom 10. bis 18. Oktober 2015 statt.

Als offenesdresden.de wollen wir die Gelegenheit nutzen, um mit Dir coole Veranstaltungen zu machen. Für uns ist "das Format" noch arg neu. Da gibt uns und dir Freiraum zur Ausgestaltung. Aber wir brauchen auch einfach deine Mithilfe. - Sag uns, dass du überhaupt Lust hast an unseren Veranstaltungen teilzunehmen! (Du musst dich ja noch nicht festlegen was du machen willst.) - Was sind deine Interessen für ein Code Week? (Das gilt auch insbesondere fü einen Tag oder ein Wochenende, wo wir dann Veranstaltungen machen.) - Hast du vielleicht eine oder mehrere bestimmte Ideen für eigene Veranstaltungen? - Möchtest und kannst du die Veranstaltung selber abhalten? - Hast du eine Idee wozu wir Veranstaltungen abhalten könnten?

Einladung

Programmcodes steuern unsere digitale Welt und damit unser tägliches Leben. Wer diese Sprache spricht, kann unsere Zukunft aktiv, individuell und kreativ mitgestalten. Darum ruft die EU in der Woche vom 10. - 18. Oktober alle Kinder, Jugendlichen und Erwachsene in ganz Europa wieder zum Programmieren auf. Auch du kannst da dabei sein.

Code Week 2015 in Dresden

Auch hier in Dresden wird es die Code Week geben. Dazu organisiert ein Zusammenschluss von Engagierten für die Wochenenden Veranstaltungen zu den Wochenenden.

Zum ersten Wochenende, 10. und 11. Oktober 2015, wird es von GeekGirlsCarrots/ erstmalig eine Veranstaltung geben. Bei der Veranstaltung unter dem Motto Code Carrots sollen insbesondere Frauen angesprochen werden.

Zum zweiten Wochenende, am 17. und 18. Oktober 2015, wird es vom Medienkulturzentrum Dresden und OK Lab Dresden Veranstaltungen für Jugendliche geben. Jugend hackt (Ost), was im Juni stattfand, hat gezeigt wie toll es jungen Menschen Code schreiben zu lassen.
Es wird aber auch von offenesdresden.de einen Hackathon zu Open Data geben. Als Open Data Dresden möchten wir über den C3D2 hinaus Menschen für das Schreiben von Code zu Open Data begeistern.

Menschen sollen während der Code Week an die Welt der Technik herangeführt werden und tiefer in sie hinein tauchen können. Jungen Menschen sollen schon früh spielerisch erlernen, dass programmieren Spaß macht. Aber auch Ältere können das Programmieren lernen und weiter vertiefen. Das OK Lab Dresden möchte daher alle Interessierten einladen Veranstaltungen in diser Woche abzuhalten, sich daran zu beteiligen oder einfach zu besuchen. Von jung bis altCode Week, egal ob jung und alt.

Eine Übersicht aller Veranstaltungen in Europa zur Code Week findet sich auf events.codeweek.eu.
Ergänzend gibt es auch noch deuschsprachige Informationen auf www.codeweek.de.
